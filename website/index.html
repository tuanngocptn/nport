Get Started View on npm

How it works

Three steps to expose your local server to the world

1

Install

One command to install globally via npm

$ npm i -g nport
2

Run

Start your tunnel with a custom subdomain

$ nport 3000 -s myapp
3

Share

Your local server is now publicly accessible

https://myapp.nport.link
